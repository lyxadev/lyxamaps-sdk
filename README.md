# LyxaMaps SDK

This is the LyxaMaps SDK for Android. It provides functionalities for interacting with the LyxaMaps API.

## Installation

Add the following dependency to your `build.gradle`:

```gradle
implementation 'com.example:lyxamaps-sdk:1.0.0'
```

## Usage

To use the LyxaMaps SDK, initialize the `LyxaMapsClient` and call the appropriate methods as needed.

## Features
- Geocoding
- Map services
- Location tracking
